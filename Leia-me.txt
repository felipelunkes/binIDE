IDE do Bin S.O para Windows NT 
---*---*---*---*---*---*---*---

Esta IDE fornece as ferramentas básicas para criar aplicativos de terceiros para o Bin S.O.

Ferramentas inclusas nesta versão:

a++: Gera um template de programa que inclui as principais bibliotecas.
binc: Compila o programa criado após a edição.

Use binc para compilar seu aplicativo.

Clique duas vezes em binc e siga os passos da tela.






Para Bin S.O 0.1.1 apenas.
