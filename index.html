IDE para desenvolvimento de aplicativos do Bin S.O

Esta IDE fornece o básico para que desenvolvedores possam criar e manter aplicativos e executáveis Bin S.O. Os aplicativos Bin S.O devem ser codificados em linguagem assembly, usando uma série de diretivas e otimizações para a correta execução pelo Bin S.O.

Componentes da IDE:

> montar - Programa para compilar/montar corretamente o aplicativo.

> instalarapp - Programa para os mais diversos fins dentro da IDE, tem a função de instalar o sistema através de arquivos em um dispositivo de armazenamento em massa, iniciar uma instância de máquina virtual para executar o sistema.

> imagem - Ferramenta para criar imagens virtuais do sistema em diversos formatos.

> suite - Oferece diversas opções para o funcionamento da IDE.

> configure - Ferramenta para instalar automaticamente o restante das dependências necessárias para o funcionamento da IDE.

> sisinfo - Exibe informações sobre o sistema operacional baseado em Linux.

> info - Exibe a versão da API do Bin S.O e da IDE do Bin S.O para Linux

> status - Checa a presença de um dispositivo candidato para a instalação do Bin S.O

> a++ - Gera templates para o desenvolvimento de aplicativos.

Realizando Download e instalando a IDE

Clique em https://github.com/felipelunkes/binIDE/archive/master/master.zip para realizar o download da IDE. À seguir, descompacte a pasta binIDE-master em um diretório qualquer em seu computador. Após a descompactação, como super-usuário (root), execute ./configure, para finalizar o restante dos processos de instalação. Você também pode realizar o download no topo desta página, mas, entratanto, os processos de instalação desta forma podem diferir da maneira descrita anteriormente.

Atualizando a IDE

A IDE oferece o serviço de atualização, em que, ao inserir ./atualizar, como super-usuário, o processo de atualização será iniciado. Todas as bibliotecas mais recentes e versões mais atualizadas de todas as ferramentas serão instaladas de forma automática.
